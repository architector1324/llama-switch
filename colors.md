# Цветовая схема (dark/light)

Источник: `static/css/style.css`.

## Основные токены темы

| Токен | Dark | Light | Где используется |
|---|---|---|---|
| `--bg-app` | `#18191a` | `#f3f4f6` | Фон приложения, фон логов (по умолчанию) |
| `--bg-card` | `#242526` | `#ffffff` | Фон панелей/карточек |
| `--bg-hover` | `#3a3b3c` | `#f4f4f5` | Hover-эффекты элементов |
| `--bg-input` | `#3a3b3c` | `#ffffff` | Фон инпутов/селектов |
| `--border` | `#3a3b3c` | `#e4e4e7` | Обычные границы |
| `--border-active` | `#505153` | `#d4d4d8` | Активные/hover границы |
| `--text-main` | `#f4f4f5` | `#18191b` | Основной текст |
| `--text-muted` | `#a1a1aa` | `#71717a` | Вторичный текст |
| `--primary` | `#3b82f6` | `#6366f1` | Primary-кнопки, акценты |
| `--primary-hover` | `#2563eb` | `#4f46e5` | Hover для primary |
| `--primary-glow` | `rgba(59, 130, 246, 0.15)` | `rgba(59, 130, 246, 0.15)` | Свечение акцента |
| `--purple` | `#8b5cf6` | `#a78bfa` | Кнопка WebUI (фиолетовый акцент) |
| `--purple-hover` | `#7c3aed` | `#8b5cf6` | Hover для фиолетового акцента |
| `--danger` | `#ef4444` | `#f87171` | Danger-кнопки |
| `--danger-hover` | `#dc2626` | `#ef4444` | Hover для danger |
| `--success` | `#10b981` | `#10b981` | Успешный статус/индикатор |
| `--theme-toggle-color` | `#38bdf8` | `#38bdf8` | Иконка переключателя темы |
| `--logo-gradient` | `linear-gradient(135deg, #fff 0%, #a1a1aa 100%)` | `linear-gradient(135deg, #18181b 0%, #52525b 100%)` | Градиент логотипа |

## Дополнительные прямые цвета (вне токенов)

| Где | Dark | Light | Примечание |
|---|---|---|---|
| `input[type="number"]` текст | `#fff` | `var(--text-main)` = `#18191b` | В light переопределяется отдельно |
| `.logs-container` фон | `var(--bg-app)` = `#18191a` | `#f9fafb` | В light есть отдельный блок `data-theme="light"` |
| `.logs-container` текст | `#d1d5db` | `#374151` | Для читаемости логов |
| `.btn-warning` фон | `#f59e0b` | `#f59e0b` | Не зависит от темы |
| Hover скроллбара `::-webkit-scrollbar-thumb:hover` | `#52525b` | `#52525b` | Не зависит от темы |

## RGBA-акценты и тени

| Где | Значение |
|---|---|
| Активный элемент модели (фон) | `rgba(16, 185, 129, 0.08)` |
| Активный элемент модели (граница) | `rgba(16, 185, 129, 0.2)` |
| Свечение индикатора online | `rgba(16, 185, 129, 0.4)` |
| Тень primary-кнопки | `rgba(59, 130, 246, 0.3)` |
| Focus-тень у number input | `rgba(59, 130, 246, 0.1)` |
| Тень WebUI-кнопки | `rgba(139, 92, 246, 0.3)` |

Если переносишь схему в другое приложение, обычно достаточно первой таблицы (основные токены темы) и пары прямых цветов для логов.
